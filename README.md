# Numerik

Konfigurasi
-------------
  ### Windows ###
    Copykan ke htdocs
    Import file sql ke database kalian
    ganti setting koneksi.php sesuai dengan local database kalian
    ganti url di app.js (js/dist/app.js) sesuai dengah PATH kalian
    
  ### Linux ###
    Copykan ke /var/www/html
    Import file sql ke database kalian
    ganti setting koneksi.php sesuai dengan local database kalian
    ganti url di app.js (js/dist/app.js) sesuai dengah PATH kalian
    

selamat menggunakan
